Modelo de caixas

  • height = Altura
  • width = Largura
  • border = Borda
  • padding = Preenchimento(Espaço interno)
  • margin = Margem(Espaço externo)
  • outline = Contrno ou Traçado

Tipos de caixas Box-level e Inline-level

Box-level Inline-level
div span
h1-h6 a
p code
main small
header strong
nav em
article sup ou sub
aside label
footer button
form input
video select

Todo elemento que seja uma box-level para centraliza-lo use margin:auto, caso for um texto use text-align:center
